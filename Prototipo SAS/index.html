¿Que es S.A.S?

School Assistence System (S.A.S) Es un sistema digital utilizado para el registro de asistencia eficiente en las instituciones

Funcionalidades

Sistema NFC

S.A.S Implementa el sistema NFC para registrar los datos de los alumnos al sistema

Camara Biometrica

Complementamos nuestro sistema con reconocedores faciales para maximizar la seguirdad y fiabilidad de los datos

Dashboard

Nuestro sistema cuenta con un panel amigable para el acceso y edicion rapida de los datos
